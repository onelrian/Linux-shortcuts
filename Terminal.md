# Navigating The Linux System From The Keyboard

 ***
 >>>Navigating the Linux system is quite intresting and this experience can be enhanced with accurate knowledge of keyboard shortcuts 

>>>### Using "Ctrl Key"
 
 ``` Ctrl + Alt +T``` 
- Launch the terminal 

```Ctrl + C```
- This is used to stop the current command or process that is running

```Ctrl + Z``` 
- This is used to put to sleep a runing program

```Ctrl + L``` 
- This is used to move to the next page giving you the ability to scroll up to your previous 
ode without necessarily deleting or using the ```clear``` command
```Ctrl + A``` 
- Moving the cursor to the start of the line (while on the terminal as cthis has a different function in visual studio)

```Ctrl + E ``` 
- This is used to move the cursor to the end of the line

``` Ctrl + D``` 
- Closing the shell (exit)

```Ctrl shift + +``` 
- This is used to zoom in hence increasing the size of charaters and images on your screen

``` Ctrl, Shift + -``` 
- This is to zoom out 

```Ctrl + shift + C``` 
- Useful for copying on the terminal

```Ctrl + shift + V``` 
- Useful for pasting on the terminal

```Ctrl + O``` 
- This is used to save ( saving contents of a doc in nano)

```Ctrl + X``` 
- This is used to exit different modes eg on a text editor like nano

``` Ctrl + shift + F``` 
- This is to search for specific words  for example when using the ```man``` command

>>>#### For the Browser 

``` Ctrl + T``` 
- Used to open a new tab on a browser


>>>### Using The "Tab Key" 

```Tab``` 
- Autocompletes the command or file if it is unique

```Tab Tab``` 
- Pressing the Tab key twice displays all commands or filenames that start with those letters

```Ctrl+ Tab``` 
- Used to move to the next open Tab on a browser


>>>### Using "Alt Key"
``` Alt + 1```
- To move to the first Tab on a terminal

```Alt+2``` 
-To  move to the second open tab on the terminal (yes you can use Alt and any other number corresponding to the actual tab you want to open)

``` Alt+h```
 -To open the manual of the command that is just being inputed
